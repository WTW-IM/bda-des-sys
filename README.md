## Development

```
$ git clone https://github.com/WTW-IM/bda-des-sys.git
$ cd bda-des-sys
$ npm install
```

Run development page on **http://localhost:3200/**

```
$ npm run dev
```

## Build

Build for production.

```
$ npm run build
```

## Deploy

Deploy to `gh-pages` branch on GitHub.

```
$ npm run deploy
```
