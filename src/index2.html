Ver. 0.0.2
  • Base:
    • Updated the "Contrast" section to have basic contrast check results for branded colors and grays;
    • Added 'Spinners' to the "Loading" section.
  • Containers:
    • Added guidance for site footers.
  • Controls:
    • Adjusted button text sizes;
    • Increased button padding;
      • Also fixed the issue of button text sitting too high within the affordance;
    • Added datepicker functionality.
    • Added slider functionality:
      • Single value;
      • Range;
      • Vertical variants of each of the above.
  • Navigation:
    • Wrapped all navigational element parent tags with <nav> tags;
    • Changed pipe character separators to CSS-rendered chevrons in Breadcrumbs;
    • Changed color of the badges in the global nav from @wtwViolet to @wtwMagenta.
